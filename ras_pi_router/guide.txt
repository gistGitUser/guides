1) настраиваем локальную сеть на роутере чтобы например пк, одноплатник и ip-камера были в одной сети
2) на дочернем роутере ставим адрес для лоакльной сети в п.1 а шлюзом ставим роутер в другой сети
3) на пк нужно произвести дополнительные настройки
   * ip лежит в локальной сети
   * шлюз/gateway - адрес роутера
   * dns сервер   - адрес роутера для доступа к сети

на ras pi нужно также настроить эти в файле dhcp.conf

для дефолтного подключения к rpi по ssh можно использовать ssh pi@[ip.adress]
для поиска доступных ip в сети nmap -sn [ip.address]/24
иногда может не подключаться по shh тогда перезапустить network/ssh службы
